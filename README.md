# gboard-kaomoji

This project adds some shortcuts to kaomojis.

### Installation

- Go to [Releases](https://www.github.com/prango71/gboard-kaomoji/releases) and download the source code(zip)
- Go to Gboard Settings
- Dictionary
- Personal Dictionary
- All languages
- Click the three-dots icon
- Click Import
- Select the Zip File
